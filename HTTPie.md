# HTTPie

- [HTTPie](https://httpie.io)

In the future, consider using [Yaak](https://yaak.app), [Bruno](https://www.usebruno.com) or [RapidAPI](https://paw.cloud).

## Install

Download and install **HTTPie** directly from the official website.

Install **HTTPie CLI** with [Homebrew](Homebrew.md).

```shell
brew install httpie
```
