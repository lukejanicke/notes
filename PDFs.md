# PDFs

- [Poppler](https://poppler.freedesktop.org)
- [OCRmyPDF (GitHub)](https://github.com/ocrmypdf/ocrmypdf) / [Documentation](https://ocrmypdf.readthedocs.io/en/latest/)
- [tesseract (GitHub)](https://github.com/tesseract-ocr/tesseract) / [Documentation](https://tesseract-ocr.github.io)
- [qpdf (GitHub)](https://github.com/qpdf/qpdf)

## Install

Install **Poppler** with [Homebrew](Homebrew.md).

```shell
brew install poppler
```

Install **OCRmyPDF** with [Homebrew](Homebrew.md).

```shell
brew install OCRmyPDF
```

Install **tesseract** with [Homebrew](Homebrew.md).

```shell
brew install tesseract
```

Install **qpdf** with [Homebrew](Homebrew.md).

```shell
brew install qpdf
```
