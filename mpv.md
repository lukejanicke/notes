# mpv

- [mpv](https://mpv.io)

[IINA](https://iina.io) is essentially a native macOS wrapper for **mpv**.
## Install

Install **mpv** with [Homebrew](Homebrew.md).

```shell
brew install mpv
```

## Usage

Play a local video file.

```shell
mpv <file>
```

Play a YouTube video.

```shell
mpv "URL"
```
