# Downloader

- [download (GitHub)](https://github.com/epoiisa/download)

## Install

```shell
mkdir -p ~/.local/bin
wget -O ~/.local/bin/download \
https://raw.githubusercontent.com/epoiisa/Icons/main/download
wget -O ~/.local/bin/download.py \
https://raw.githubusercontent.com/epoiisa/Icons/main/download.py
chmod +x ~/.local/bin/download
```
